# dwm

## Installation

```sh
make clean install
```

Create a `.desktop` file at `/usr/share/xsessions`.
```sh
[Desktop Entry]
Name=dwm
Comment=suckless dwm
Exec=/home/michael/scripts/startdwm
Type=Application
```
